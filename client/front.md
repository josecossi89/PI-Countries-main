## Dependencias a instalar

- Se deben instalar las siguientes dependencias en la carpeta "client":

```
- npm react-router-dom
- npm react-redux
- npm redux
- npm redux-thunk
- npm axios
```

Se deben crear las siguientes carpetas y archivos en la carpeta "src" de "client":

```
- actions
 -
- components
- reducer
- store
   -index.js

```
